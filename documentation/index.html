Installing Template

After unzip the download pack, you'll found a Template Folder with all the files.

You can view this Template in any browser, you can display or edit the Template without an internet connection.

The three sections that will not work is the Google map Section which contains external link to get the map source.

the Contact form which uses PHP, so will work only online.

and the Twitter Feed which is avaliable only online.

Now open your FTP Client (like Filezilla) or directly through cpanel, upload the content of the Template on your server root.

Once the files are done uploading go to www.yourdomainname.com/index.html

Please follow the tutorial to have a simple installation of the theme.

Tutorial

Please follow the first steps of this tutorial, to have a better knowledge how to use the theme, without any problem
The Template has different header styles, and different patterns and of course 5 skins, we will explain how to use every skin and every header style so you don't have any trouble with the theme

Action Bar

In the demo, when opening the style switcher, u will see the option for action bar to be shown, or hide...by default the action bar is hidden, but if u want that action bar to be shown, just add this line of code in the end of a file called custom.css in css folder. 

	#Action_bar {
		display: block !important;
	}

Page Layout

The Cake Template, gives u the opportunity to have a layout of the page full width, or boxed.

If u want a full width layout, u just need to add this class "layout-full-width" into the body tag, like this : 
	<body class="home page page-parent page-template-default template-slider color-custom header-dark header-alpha sticky-header layout-full-width">
But if you want the boxed Layout, u need to add this class "layout-boxed" into the body tag, like this: 
 
	<body class="home page page-parent page-template-default template-slider color-custom header-dark header-alpha sticky-header layout-boxed">


Patterns for Boxed Layout

We included our 20 patterns that u can see in demo, but u can use whatever pattern u want. To use any of our paterns go to custom.css, in the end of the file add this line of code: 
 
	html {
	  background: url(../images/patterns/1.png) !important;
	}
U can choose whichever patern u want from the pattern palete which can be found in folder images, patterns folder.
Note:The patterns are only for the Boxed layout

Headers

There are a 4 different header styles which we will explain for each of them how to use. The first header is the white background header. If u want this header u can use it by adding these classes "header-white" into the body tag, like this: 
 
	<body class="home page page-parent page-template-default template-slider color-custom sticky-header layout-boxed header-white header-bg">
Note: The white header, will make the navigation also in white

The second header is Dark Background header... just add this class in the body tag "header-dark": 
 
	<body class="home page page-parent page-template-default template-slider color-custom sticky-header layout-boxed header-dark header-bg">
Note: The dark header, will make the navigation also in dark

The third header is the transparent bg with the nav background of white, and the color of black, to use this add these classes in the body tag "header-white header-alpha": 
 
	<body class="home page page-parent page-template-default template-slider color-custom sticky-header layout-boxed header-white header-alpha">

The last one, but not least is the transparent header with the navigation with dark background and white color. To use this header ad these classes to the body tag "header-dark header-alpha": 
 
	<body class="home page page-parent page-template-default template-slider color-custom sticky-header layout-boxed header-dark header-alpha">
By default the headers are sticky, but if u want them to be in static position, just remove this class "sticky-header" from the body tag and u have a static header.
